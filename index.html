Super simples! Você faz uma requisição HTTP POST para nossa API com o texto do cliente. Em 0.3s recebe o perfil detectado. Temos código pronto para ManyChat, Dialogflow, Typebot, Make.com e mais.

Depende do seu volume. Em média: 100 leads/mês = 100 créditos. 500 leads/mês = 500 créditos. O plano Starter (2.000 créditos) dura ~2 meses para a maioria dos lançamentos.

Não! PSI funciona com qualquer chatbot. É só uma consulta HTTP que você adiciona no fluxo. Seu chatbot continua igual, só fica mais inteligente.
